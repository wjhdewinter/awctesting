# AWC Warehouse Tool — Versie 1.0 / V25 PRO Business

Mobiele webapp voor Amsterdam Warehouse Company voor dagelijkse warehouse-registraties.

## Belangrijkste modules
- Pallets tellen
- Personeel checken inclusief roosterfoto OCR
- Warehouse ruimtechecks per locatie/ruimte
- Order wegen en meten per colli/pallet
- Meerdere artikelnummers per colli/pallet
- Foto's en bestanden toevoegen in de module Order wegen
- Machinechecks
- Incidenten
- Schade en manco registreren

## V25 PRO Business updates
- Meerdere ontvangers selecteren bij mail
- CC en BCC velden toegevoegd
- Nieuwe ontvangers: Mira en Order
- Slimme mailonderwerpen per module
- PDF's zakelijker opgemaakt met AWC-logo/blok, bedrijfsnaam en locaties
- Foto's in PDF middelgroot en beter leesbaar
- Excel export met betere kolombreedtes en autofilter
- Module Order wegen ondersteunt foto's en PDF-bestanden voor klantcommunicatie

## Installatie
Upload alle bestanden naar GitHub Pages. Open de app via HTTPS en gebruik daarna de knop **Install** of het browsermenu **App installeren / Toevoegen aan startscherm**.

## Let op over e-mailbijlagen
Een browser kan via `mailto:` geen bestanden automatisch als bijlage meesturen of direct aan een bestaande e-mailthread koppelen. De app opent een mailconcept met ontvangers, CC/BCC, onderwerp en tekst. Voeg daarna de gegenereerde PDF/foto's handmatig toe in Outlook/Gmail.
