Handleiding - AWC Warehouse Tool versie 1.0

Deze app is bedoeld voor dagelijkse controles en registraties op de werkvloer. Vul per module de gegevens in, voeg waar nodig foto’s toe en gebruik daarna Opslaan, PDF, Mail of Excel.

Algemeen gebruik

  1. Kies links in het menu de juiste module.
  2. Vul gebruiker, locatie en eventuele bijzonderheden in.
  3. Kies bij mail de ontvanger. In de app zie je alleen de naam van de ontvanger.
  4. Gebruik Opslaan om de registratie in de historie te bewaren.
  5. Gebruik PDF, Mail, Excel of WhatsApp om de registratie te delen.

Modules

  • Pallets: tel Euro, Blok, 2 meter en DPB pallets.
  • Personeel: voeg namen handmatig toe of scan een roosterfoto. Kies per persoon aanwezig, verlof, ziek, te laat of afwezig.
  • Warehouse: kies de juiste ruimte. De vragen zijn aangepast per locatie/ruimte.
  • Order wegen: vul per colli of pallet gewicht, lengte, breedte, hoogte en meerdere artikelnummers met aantallen in.
  • Machines: voer machinechecks uit. Kritieke fouten geven automatisch een waarschuwing of afkeuring.
  • Incidenten: registreer incidenten of bijna-ongevallen met oorzaak, actie en foto’s.
  • Schade en Manco: registreer artikel, aantal, locatie, toelichting en foto’s. Mailfunctie is beschikbaar.

Installeren

Open de app via GitHub Pages of HTTPS. Gebruik daarna de knop Install of het browsermenu: App installeren / Toevoegen aan startscherm.

User guide - AWC Warehouse Tool version 1.0

This app is made for daily workplace checks and registrations. Complete the fields in each module, add photos when needed, then use Save, PDF, Mail or Excel.

General use

  1. Select the correct module from the menu.
  2. Enter user, location and any remarks.
  3. Select a mail recipient. The app only shows recipient names.
  4. Use Save to store the registration in history.
  5. Use PDF, Mail, Excel or WhatsApp to share the registration.

Modules

  • Pallets: count Euro, Block, 2 meter and DPB pallets.
  • Staff: add names manually or scan a schedule photo. Set each person to present, leave, sick, late or absent.
  • Warehouse: select the correct area. Questions are adjusted per location/area.
  • Order weighing: enter weight, length, width, height and multiple item numbers with quantities per package or pallet.
  • Machines: perform machine checks. Critical failures automatically trigger warnings or rejection.
  • Incidents: register incidents or near misses with cause, action and photos.
  • Damage and Shortage: register item, quantity, location, remarks and photos. Mail is available.

Installation

Open the app through GitHub Pages or HTTPS. Then use the Install button or browser menu: Install app / Add to home screen.
